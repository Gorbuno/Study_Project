1. Дополнить программу, разработанную в ходе выполнения лабораторной работы №1, выполняющую сортировку массива методом выбора:
	алгоритмом поиска произвольного целого числа методом двоичного поиска

2. Разработать программу, осуществляющую для файла “ads_lab2.dat” (описание формата приведено далее): 
	построение двоичного дерева поиска для столбца «Цена»
	вывод статистики по структуре дерева
	поиск элементов с его помощью (после ввода цены с консоли должны быть выведены все записи с искомой ценой либо сообщение об отсутствии искомых)

3. Разработать программу, осуществляющую для файла “ads_lab2.dat” (описание формата приведено далее): 
	построение хеш-таблицы для столбца «Артикул» (в качестве учебной хеш-функции взять остаток от деления числа на 35129)
	вывод статистики по хеш-таблице

При отображении найденных записей на экран выводятся все три поля: артикул, цена, производитель (например, через пробел). Каждая найденная запись выводится в отдельной строке.
